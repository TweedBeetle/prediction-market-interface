`fastmcp.utilities.mcp_server_config.v1.mcp_server_config`
=====================================================================================================================================================================================================================

FastMCP Configuration File Support. This module provides support for fastmcp.json configuration files that allow users to specify server settings in a declarative format instead of using command-line arguments.
