`fastmcp.server.auth.providers.github`
=========================================================================================================================================================

GitHub OAuth provider for FastMCP. This module provides a complete GitHub OAuth integration that’s ready to use with just a client ID and client secret. It handles all the complexity of GitHub’s OAuth flow, token validation, and user management. Example:

Copy

    from fastmcp import FastMCP
    from fastmcp.server.auth.providers.github import GitHubProvider
    
    # Simple GitHub OAuth protection
    auth = GitHubProvider(
        client_id="your-github-client-id",
        client_secret="your-github-client-secret"
    )
    
    mcp = FastMCP("My Protected Server", auth=auth)

`GitHubProvider`[](https://github.com/jlowin/fastmcp/blob/main/src/fastmcp/server/auth/providers/github.py#L170)

Complete GitHub OAuth provider for FastMCP. This provider makes it trivial to add GitHub OAuth protection to any FastMCP server. Just provide your GitHub OAuth app credentials and a base URL, and you’re ready to go. Features:

*   Transparent OAuth proxy to GitHub
*   Automatic token validation via GitHub API
*   User information extraction
*   Minimal configuration required
